## 1) Intercepting HTTP request and responses

- Clear enough, proxy tab then interception on and then forwad as the page loads
- In options tab, you can decide what to intercep and what all things to skip, and can even add your own rules and enable it to intercept responses

## 2)Burp Repeater

- Used to repeat individual requests
- Right click on the proxy with the current request and send to repeater or just do the same on the history
- Can do it from the target tab in proxy too
- Can work with redirecrtions too, can also have a setting to follow redirections too from the repeater menu
- each tab has individual repeater tabs so you can work on multiple requests
- You can copy a url, right click on a new repeater tabs request area and then paste url as request
